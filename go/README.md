# exercises / go

Упражнения по Go. Публичный репозиторий. Монтируется в контейнер как `/exercises/go` (read-only).

## Структура

```
go/
├── exercises.json          — каталог упражнений
├── component.toml          — манифест компонента
├── scripts/
│   └── session.sh          — инициализация сессии в контейнере
├── go-expr-01/             — Таблица умножения (easy)
├── go-func-01/             — Реверс строки (easy)
└── go-multi-01/            — Пакет mathutils (medium)
```

Каждое упражнение содержит:
- `task.md` — условие задачи
- `check.sh` — скрипт автопроверки (команда `check` в терминале)
- `workspace/` — стартовые файлы, копируются в `~/exercise/` при старте сессии

## Добавление нового упражнения

1. Создать папку `go-<тип>-<номер>/`
2. Добавить `task.md`, `check.sh`, `workspace/`
3. Зарегистрировать в `exercises.json`
